tcfig.main(ctx=None, c_config_file='config.toml', c_check=True, c_post=False)

Main function that controls the running of the program

Parameters

  • ctx – Click context

  • c_config_file (bool, optional) – Function parameter for the config file path. This is not used if the context is provided (when the CLI is not used, for exemple in a python console)

  • c_check (bool, optional) – Function parameter for the existence check This is not used if the context is provided (when the CLI is not used, for exemple in a python console)

  • c_post (bool, optional) – Function parameter for the CloudFlare’s post This is not used if the context is provided (when the CLI is not used, for exemple in a python console)
